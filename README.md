# Glimpse-o-Matic

A lightweight browser-based photo gallery viewer. As a bonus, it watermarks
images with client information.

This started as a human-written project in JavaScript, with the intent to
migrate to Rust/WASM at some point. I worked on it every year or two for a while, but didn't make a ton of progress. Then years after inception, 
along came Claude Code. I've used Claude to actualize several hobby
projects, including this one, and so far I think the results have been
good. For sure, none of this would have been done by now, and, if I'm being
honest, probably ever.

**[Open Glimpse-o-Matic](https://remidodoso.github.io/glimpse-o-matic/)**

---

## GitHub Pages setup

1. Go to **Settings → Pages** in this repo
2. Under *Build and deployment*, set Source to **Deploy from a branch**
3. Set Branch to `master`, folder `/` (root)
4. Click **Save** — the site will be live at the link above within a minute or two
